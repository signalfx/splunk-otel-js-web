4 Media & Features features.canvas · features.video · features.iframes · features.packAssets

Canvas animation

Requires features.canvas enabled in Settings ⚙. Bouncing-ball animation drawn via requestAnimationFrame.

Video element

Requires features.video enabled in Settings ⚙. Play, pause, and seek to generate video playback events in the recording.

Iframe content

Requires features.iframes enabled in Settings ⚙. Same-origin iframe — the recorder captures its DOM separately.

Image assets

Enable features.packAssets in Settings ⚙ to include image data inline in the recording. Mix of data-URI and HTTP sources.
